
# Egg Timer

### Egg Timer is an application to countdown how long an egg will be boiled.

What's in this app:
- Usage of UIProgressView to keep track of time.
- Usage of Swift control flow: If/Else and Switch Statements.
- Usage of Swift Dictionaries.
- Usage of Swift Optionals.
